¿Qué es el método del Trapecio?

El método del Trapecio aproxima el área bajo una curva dividiendo el intervalo en varios trapecios. En nuestro caso, como tenemos datos diarios (30 días), dividimos el mes en 30 trapecios, cada uno con ancho h = 1 día.

Área de un trapecio

El área de un trapecio se calcula como el promedio de las alturas (los precios) multiplicado por el ancho:

Área del trapecio i = (h/2) × [Precio(día i) + Precio(día i+1)]

Donde h = 1 día es el ancho de cada trapecio.

Aplicación a nuestro problema

Para calcular el gasto total del mes, sumamos las áreas de los 30 trapecios:

Gasto total = Σ [ (1/2) × (Precio_día_i + Precio_día_i+1) ]

Simplificando, obtenemos la fórmula final donde los precios intermedios (días 2 al 29) se multiplican por 2 porque cada uno pertenece a dos trapecios:

Trapecio = (1/2) × [P₁ + 2P₂ + 2P₃ + ... + 2P₂₉ + P₃₀]

¿Qué significa este resultado?

Comparativa con Simpson 1/3: El método del Trapecio tiene un error de O(h²) = 1/12 ≈ 0.0833, mientras que Simpson 1/3 tiene error O(h⁴) = 1/90 ≈ 0.0111, siendo este último más preciso.
