Método Numérico - Heun

📈 Método de Heun

Método Predictor-Corrector aplicado al modelo social

📈 Fórmula: Nₙ₊₁ = Nₙ + (h/2)[f(tₙ,Nₙ) + f(tₙ₊₁, Nₙ + h·f(tₙ,Nₙ))]

PARTE 1: ¿Por qué necesitamos un método mejor que Euler?

El problema de Euler en modelos sociales

🚗 Analogía: Imagina que manejas un auto y quieres saber dónde estarás en 1 hora.

  • Euler: Solo mira la velocidad al inicio de la hora. Si aceleras o frenas durante la hora, ¡tu cálculo será incorrecto!
  • Heun: Mira la velocidad al inicio Y al final de la hora, y usa el promedio. ¡Mucho más preciso!

En nuestro modelo social, las tasas de cambio (dN/dt, dM/dt, dD/dt) NO son constantes. Cambian día a día a medida que evolucionan N, M y D. Por eso necesitamos un método más preciso.

PARTE 2: ¿Cómo funciona el método de Heun?

Predictor-Corrector: El método en DOS pasos

PASO 1: PREDICTOR (usando Euler)

Primero, hacemos una predicción usando el método de Euler:

Npred = Nₙ + h × (-a·Nₙ·Mₙ + b·Dₙ)
Mpred = Mₙ + h × (a·Nₙ·Mₙ - c·Mₙ·Dₙ)
Dpred = Dₙ + h × (k·Mₙ - r·Dₙ)

Esto nos da una primera aproximación de cómo serán las poblaciones mañana.

PASO 2: CORRECTOR (promediando pendientes)

Luego, corregimos usando el PROMEDIO de las pendientes:

Nₙ₊₁ = Nₙ + (h/2) × (pendiente_inicial_N + pendiente_predicha_N)
Mₙ₊₁ = Mₙ + (h/2) × (pendiente_inicial_M + pendiente_predicha_M)
Dₙ₊₁ = Dₙ + (h/2) × (pendiente_inicial_D + pendiente_predicha_D)

¡Esto es como usar la velocidad PROMEDIO en lugar de solo la velocidad inicial!

La fórmula completa de Heun para el modelo social

Para Neutrales (N):
k₁_N = -a·Nₙ·Mₙ + b·Dₙ
N_pred = Nₙ + h·k₁_N
k₂_N = -a·N_pred·M_pred + b·D_pred
Nₙ₊₁ = Nₙ + (h/2)·(k₁_N + k₂_N)
Para Manifestantes (M):
k₁_M = a·Nₙ·Mₙ - c·Mₙ·Dₙ
M_pred = Mₙ + h·k₁_M
k₂_M = a·N_pred·M_pred - c·M_pred·D_pred
Mₙ₊₁ = Mₙ + (h/2)·(k₁_M + k₂_M)
Para Mediadores (D):
k₁_D = k·Mₙ - r·Dₙ
D_pred = Dₙ + h·k₁_D
k₂_D = k·M_pred - r·D_pred
Dₙ₊₁ = Dₙ + (h/2)·(k₁_D + k₂_D)

PARTE 3: Ejemplo práctico paso a paso

Aprendamos con números reales

Datos de ejemplo:

  • Neutrales iniciales (N₀) = 900 personas
  • Manifestantes iniciales (M₀) = 100 personas
  • Mediadores iniciales (D₀) = 50 personas
  • Parámetros: a = 0.008, b = 0.1, c = 0.008, k = 0.3, r = 0.2
  • Paso de tiempo h = 1 día

📌 PASO 1: PREDICTOR (usando Euler)

Calculamos las pendientes iniciales:

k₁_N = -0.008×900×100 + 0.1×50 = -720 + 5 = -715

k₁_M = 0.008×900×100 - 0.008×100×50 = 720 - 40 = +680

k₁_D = 0.3×100 - 0.2×50 = 30 - 10 = +20

Predecimos los valores para mañana:

N_pred = 900 + 1×(-715) = 185

M_pred = 100 + 1×(680) = 780

D_pred = 50 + 1×(20) = 70

📌 PASO 2: CORRECTOR (promediando pendientes)

Calculamos las pendientes con los valores predichos:

k₂_N = -0.008×185×780 + 0.1×70 = -1154.4 + 7 = -1147.4

k₂_M = 0.008×185×780 - 0.008×780×70 = 1154.4 - 436.8 = +717.6

k₂_D = 0.3×780 - 0.2×70 = 234 - 14 = +220

Promediamos las pendientes:

Pendiente promedio N = (-715 + -1147.4)/2 = -931.2

Pendiente promedio M = (680 + 717.6)/2 = +698.8

Pendiente promedio D = (20 + 220)/2 = +120

Valores corregidos (resultado final):

N₁ = 900 + 1×(-931.2) = -31.2 → 0 personas (no puede ser negativo)

M₁ = 100 + 1×(698.8) = 798.8 personas

D₁ = 50 + 1×(120) = 170 personas

PARTE 4: Comparación: Euler vs Heun

📐 Euler

Usa 1 pendiente (solo al inicio)

Error: O(h) = 0.1

✅ Simple y rápido

❌ Menos preciso

❌ No captura cambios bruscos

📈 Heun

Usa 2 pendientes (inicio y final)

Error: O(h²) = 0.01

✅ Más preciso

✅ Captura mejor los cambios

❌ Requiere 2 cálculos por paso

Heun es 10 VECES más preciso que Euler (0.01 vs 0.1)

PARTE 6: Ventajas y desventajas de Heun en este modelo

Ventajas

  • ✅ 10 veces más preciso que Euler
  • ✅ Captura mejor los cambios bruscos en la dinámica social
  • ✅ Ideal para modelos no lineales como este
  • ✅ Buen balance entre precisión y complejidad

Desventajas

  • ❌ Requiere el doble de cálculos que Euler
  • ❌ No tan preciso como RK4
  • ❌ Puede ser más lento para simulaciones largas

¿Cuándo usar Heun en este modelo?
Heun es el mejor balance entre precisión y simplicidad para el modelo de difusión social. Es especialmente útil cuando:
  • 📌 Se necesita más precisión que Euler pero no se requiere la máxima de RK4
  • 📌 El modelo tiene dinámicas no lineales (como N×M)
  • 📌 Se quiere simular rápidamente con buena exactitud

Para decisiones críticas (como predecir el estallido de protestas), se recomienda usar RK4.
