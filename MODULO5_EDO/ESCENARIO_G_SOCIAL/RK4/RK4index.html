Método Numérico - Runge-Kutta 4

⭐ Método de Runge-Kutta 4 (RK4)

El método más preciso para simular la dinámica social

⭐ Fórmula: yₙ₊₁ = yₙ + (h/6)(k₁ + 2k₂ + 2k₃ + k₄)

PARTE 1: ¿Por qué necesitamos el método más preciso?

La evolución de los métodos numéricos

📐 Euler

1 pendiente

Error: O(h)

Precisión: ⭐

Solo mira al inicio

📈 Heun

2 pendientes

Error: O(h²)

Precisión: ⭐⭐

Promedia inicio y final

⭐ RK4

4 pendientes

Error: O(h⁴)

Precisión: ⭐⭐⭐⭐⭐

RK4 es 10,000 VECES más preciso que Euler (para h=0.1, error de Euler ~0.1, error de RK4 ~0.0001)

En modelos sociales donde pequeños cambios pueden desencadenar crisis, ¡esta precisión es CRUCIAL!

PARTE 2: ¿Cómo funciona el método RK4?

La idea: Calcular 4 pendientes en diferentes puntos

RK4 calcula CUATRO pendientes en diferentes puntos del intervalo y las promedia con pesos específicos:

📌 k₁ = f(tₙ, Nₙ, Mₙ, Dₙ)
Pendiente al INICIO del intervalo
📌 k₂ = f(tₙ + h/2, Nₙ + (h/2)·k₁_N, Mₙ + (h/2)·k₁_M, Dₙ + (h/2)·k₁_D)
Pendiente en el PUNTO MEDIO (usando k₁)
📌 k₃ = f(tₙ + h/2, Nₙ + (h/2)·k₂_N, Mₙ + (h/2)·k₂_M, Dₙ + (h/2)·k₂_D)
Pendiente en el PUNTO MEDIO (usando k₂)
📌 k₄ = f(tₙ + h, Nₙ + h·k₃_N, Mₙ + h·k₃_M, Dₙ + h·k₃_D)
Pendiente al FINAL del intervalo

La fórmula completa de RK4 para el modelo social

Para CADA una de las tres variables (N, M, D):

yₙ₊₁ = yₙ + (h/6) × (k₁ + 2k₂ + 2k₃ + k₄)

Los pesos son: 1, 2, 2, 1 - ¡da más peso a las pendientes del punto medio (k₂ y k₃)!

Para Neutrales (N):
Nₙ₊₁ = Nₙ + (h/6) × (k₁_N + 2k₂_N + 2k₃_N + k₄_N)

Para Manifestantes (M):
Mₙ₊₁ = Mₙ + (h/6) × (k₁_M + 2k₂_M + 2k₃_M + k₄_M)

Para Mediadores (D):
Dₙ₊₁ = Dₙ + (h/6) × (k₁_D + 2k₂_D + 2k₃_D + k₄_D)

PARTE 3: Ejemplo práctico paso a paso

Aprendamos con números reales

Datos de ejemplo:

  • Neutrales iniciales (N₀) = 900 personas
  • Manifestantes iniciales (M₀) = 100 personas
  • Mediadores iniciales (D₀) = 50 personas
  • Parámetros: a = 0.008, b = 0.1, c = 0.008, k = 0.3, r = 0.2
  • Paso de tiempo h = 1 día

📌 Función del sistema (para recordar):

dN/dt = -a·N·M + b·D

dM/dt = a·N·M - c·M·D

dD/dt = k·M - r·D

📌 PASO 1: Calculamos k₁ (pendiente al inicio)

k₁_N = -0.008×900×100 + 0.1×50 = -720 + 5 = -715

k₁_M = 0.008×900×100 - 0.008×100×50 = 720 - 40 = +680

k₁_D = 0.3×100 - 0.2×50 = 30 - 10 = +20

📌 PASO 2: Calculamos k₂ (pendiente en el punto medio, usando k₁)

Valores en el punto medio:

N_mid = 900 + (1/2)×(-715) = 900 - 357.5 = 542.5

M_mid = 100 + (1/2)×(680) = 100 + 340 = 440

D_mid = 50 + (1/2)×(20) = 50 + 10 = 60

k₂_N = -0.008×542.5×440 + 0.1×60 = -1909.6 + 6 = -1903.6

k₂_M = 0.008×542.5×440 - 0.008×440×60 = 1909.6 - 211.2 = +1698.4

k₂_D = 0.3×440 - 0.2×60 = 132 - 12 = +120

📌 PASO 3: Calculamos k₃ (pendiente en el punto medio, usando k₂)

Valores en el punto medio (usando k₂):

N_mid2 = 900 + (1/2)×(-1903.6) = 900 - 951.8 = -51.8 → 0

M_mid2 = 100 + (1/2)×(1698.4) = 100 + 849.2 = 949.2

D_mid2 = 50 + (1/2)×(120) = 50 + 60 = 110

k₃_N = -0.008×0×949.2 + 0.1×110 = 0 + 11 = +11

k₃_M = 0.008×0×949.2 - 0.008×949.2×110 = 0 - 835.3 = -835.3

k₃_D = 0.3×949.2 - 0.2×110 = 284.8 - 22 = +262.8

📌 PASO 4: Calculamos k₄ (pendiente al final, usando k₃)

Valores al final:

N_end = 900 + 1×(+11) = 911

M_end = 100 + 1×(-835.3) = -735.3 → 0

D_end = 50 + 1×(262.8) = 312.8

k₄_N = -0.008×911×0 + 0.1×312.8 = 0 + 31.3 = +31.3

k₄_M = 0.008×911×0 - 0.008×0×312.8 = 0 - 0 = 0

k₄_D = 0.3×0 - 0.2×312.8 = 0 - 62.6 = -62.6

📌 PASO 5: Aplicamos la fórmula RK4

Para Neutrales (N):

N₁ = 900 + (1/6)×[(-715) + 2×(-1903.6) + 2×(11) + 31.3]

N₁ = 900 + (1/6)×[-715 - 3807.2 + 22 + 31.3] = 900 + (1/6)×[-4468.9] = 900 - 744.8 = 155.2 personas

Para Manifestantes (M):

M₁ = 100 + (1/6)×[680 + 2×(1698.4) + 2×(-835.3) + 0]

M₁ = 100 + (1/6)×[680 + 3396.8 - 1670.6 + 0] = 100 + (1/6)×[2406.2] = 100 + 401 = 501 personas

Para Mediadores (D):

D₁ = 50 + (1/6)×[20 + 2×(120) + 2×(262.8) + (-62.6)]

D₁ = 50 + (1/6)×[20 + 240 + 525.6 - 62.6] = 50 + (1/6)×[723] = 50 + 120.5 = 170.5 personas

PARTE 4: Comparación: Euler vs Heun vs RK4

¿Por qué RK4 es el MEJOR para este modelo?
El modelo de difusión social es altamente no lineal (tiene términos como N×M). Esto significa que pequeños errores se amplifican rápidamente. RK4, con su error O(h⁴), minimiza estos errores.

PARTE 6: Ventajas y desventajas de RK4

Ventajas

  • ✅ Máxima precisión (error O(h⁴))
  • ✅ 10,000 veces más preciso que Euler
  • ✅ Ideal para modelos no lineales como la difusión social
  • ✅ Es el estándar en ingeniería y ciencias
  • ✅ Captura perfectamente los cambios bruscos

Desventajas

  • ❌ Requiere 4 evaluaciones por paso (más costoso)
  • ❌ Más complejo de implementar
  • ❌ Puede ser más lento para simulaciones largas
