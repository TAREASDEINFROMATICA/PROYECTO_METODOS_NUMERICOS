Modulo 3 - Interpolacion

Metodo de Splines Cubicos

El metodo de Splines Cubicos permite estimar precios faltantes usando polinomios cubicos por tramos. Es muy util cuando se busca una curva suave y mas estable visualmente.

Sᵢ(x) = aᵢ + bᵢ(x-xᵢ) + cᵢ(x-xᵢ)² + dᵢ(x-xᵢ)³

Cada tramo une dos puntos conocidos y mantiene continuidad en la curva, evitando saltos bruscos entre precios.

Contexto del ejercicio

¿Que estamos calculando?

En este ejercicio se analiza la variacion de precios de productos basicos durante una crisis. Como no se tienen datos para todos los dias, se usan puntos conocidos para construir una curva suave de precios.

A diferencia de Lagrange y Newton, Splines Cubicos no usa un solo polinomio grande, sino varios polinomios cubicos por intervalos.

Datos conocidos

Son pares ordenados de la forma (dia, precio).

Tramos

Interpretacion de variables

Variables usadas en Splines Cubicos

Caso I: Estado ideal

En este caso se analiza un mercado estable con precios que suben lentamente.

Funcion del modelo: Sᵢ(x) = aᵢ + bᵢ(x-xᵢ) + cᵢ(x-xᵢ)² + dᵢ(x-xᵢ)³

Variables del metodo

  • x: dia donde se quiere estimar el precio.
  • xᵢ: dia inicial de cada tramo.
  • yᵢ: precio conocido en el dia xᵢ.
  • Sᵢ(x): polinomio cubico del tramo i.
  • aᵢ: valor base del precio en el tramo.
  • bᵢ: coeficiente lineal.
  • cᵢ: coeficiente cuadratico.
  • dᵢ: coeficiente cubico.
  • hᵢ: distancia entre dos dias consecutivos.

Interpretacion final

Splines Cubicos construye una curva suave usando polinomios por tramos.

Interpretacion del metodo

¿Que tan estable y util fue Splines Cubicos?

Splines Cubicos fue util porque permite construir una curva mas suave entre los datos conocidos, evitando oscilaciones fuertes que pueden aparecer cuando se usa un solo polinomio grande.

Estabilidad

Es el metodo mas estable visualmente, porque trabaja por tramos y no fuerza todos los datos dentro de un solo polinomio grande.

Utilidad

Es util para representar curvas de precios mas realistas y suaves durante el mes.

Limitacion

Su procedimiento es mas largo porque necesita calcular varios coeficientes para cada tramo.

Conclusion

Splines Cubicos es el metodo mas conveniente cuando se desea una curva suave y visualmente estable. Para este modulo, es muy util porque los precios cambian por etapas durante la crisis.
