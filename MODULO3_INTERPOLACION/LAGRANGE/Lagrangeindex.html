Modulo 3 - Interpolacion

Metodo de Interpolacion de Lagrange

El metodo de Lagrange permite estimar precios faltantes usando datos conocidos de dias anteriores. En este modulo se aplica a precios de alimentos durante una crisis.

P(x) = Σ yᵢ · Lᵢ(x)

Cada Lᵢ(x) es un polinomio base que usa todos los puntos conocidos para calcular el precio aproximado en un dia sin registro.

Contexto del ejercicio

¿Que estamos calculando?

En una situacion de crisis, no siempre se tiene el precio de un producto para todos los dias. Por eso se usan datos dispersos y se estima el precio faltante mediante interpolacion.

En este caso, x representa el dia del mes y y representa el precio del producto en bolivianos. El objetivo es calcular P(x), que es el precio aproximado en el dia que no tiene dato.

Dato conocido

Es un par (dia, precio) registrado durante el mes.

Dato faltante

Es el precio que se desea estimar en un dia sin registro.

Resultado

P(x) representa el precio interpolado por el metodo de Lagrange.

Interpretacion de variables

Variables usadas en el metodo de Lagrange

Caso I: Estado ideal

En este caso se analiza un mercado estable con precios que suben lentamente.

Funcion del modelo: P(x) = Σ yᵢ · Lᵢ(x)

Variables del metodo

  • x: dia donde se quiere estimar el precio.
  • xᵢ: dia conocido dentro de los datos.
  • yᵢ: precio conocido correspondiente al dia xᵢ.
  • Lᵢ(x): polinomio base de Lagrange.
  • P(x): precio interpolado en el dia x.
  • n: cantidad de datos conocidos.
  • yᵢ · Lᵢ(x): aporte de cada dato al resultado final.

Interpretacion final

Lagrange combina todos los puntos conocidos para estimar el precio faltante.

Interpretacion del metodo

¿Que tan estable y util fue Lagrange?

El metodo de Lagrange fue util porque permite calcular directamente el precio aproximado en un dia sin construir una tabla de diferencias.

Utilidad

Es practico cuando se quiere estimar un valor puntual usando datos conocidos.

Ventaja

No necesita derivadas ni sistemas de ecuaciones. Solo usa los puntos conocidos.

Limitacion

Con muchos puntos puede generar oscilaciones. Por eso se compara despues con Newton y Splines.

Conclusion

Lagrange es muy util para estimaciones puntuales de precios. Sin embargo, para representar una curva mas suave, Splines puede ser mas estable visualmente.
