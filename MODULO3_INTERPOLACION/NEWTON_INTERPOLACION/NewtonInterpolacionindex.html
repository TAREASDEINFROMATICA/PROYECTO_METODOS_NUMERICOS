Modulo 3 - Interpolacion

Metodo de Newton por Diferencias Divididas

Este metodo permite estimar precios faltantes usando una tabla de diferencias divididas. Es muy util cuando se trabaja con datos dispersos.

P(x) = a0 + a1(x-x0) + a2(x-x0)(x-x1) + ...

Los coeficientes a0, a1, a2, ... se obtienen mediante diferencias divididas a partir de los datos conocidos.

Contexto del ejercicio

¿Que estamos calculando?

En este ejercicio se analiza el precio de productos basicos durante una crisis. Como no se tienen datos para todos los dias, se usa interpolacion para estimar precios faltantes.

En Newton Interpolacion, los datos conocidos se organizan en una tabla de diferencias divididas. Luego se construye un polinomio que permite estimar el precio en un dia especifico.

Interpretacion de variables

Variables usadas en Newton Interpolacion

Caso I: Estado ideal

En este caso se analiza un mercado estable con precios que suben lentamente.

Funcion del modelo: P(x) = a0 + a1(x-x0) + a2(x-x0)(x-x1) + ...

Variables del metodo

  • x: dia donde se quiere estimar el precio.
  • xᵢ: dia conocido dentro de los datos.
  • yᵢ: precio conocido correspondiente al dia xᵢ.
  • a0, a1, a2: coeficientes obtenidos por diferencias divididas.
  • f[xᵢ,xⱼ]: diferencia dividida entre dos puntos.
  • P(x): precio interpolado en el dia x.
  • n: cantidad de datos conocidos.

Interpretacion final

Newton usa diferencias divididas para construir el polinomio interpolante.

Interpretacion del metodo

¿Que tan estable y util fue Newton Interpolacion?

El metodo de Newton Interpolacion fue util porque organiza los datos mediante diferencias divididas y permite construir el polinomio de forma progresiva.

Utilidad

Es practico cuando se desea estimar precios faltantes y tambien cuando se quiere mostrar el proceso ordenado en una tabla.

Ventaja

Si se agrega un nuevo dato, no es necesario rehacer todo desde cero como puede ocurrir con otros metodos.

Limitacion

Igual que Lagrange, si se usan muchos puntos puede generar oscilaciones en la curva.

Conclusion

Newton Interpolacion es util y ordenado para trabajar con datos dispersos. Es muy parecido a Lagrange en resultado, pero su tabla de diferencias divididas facilita el analisis del procedimiento.
