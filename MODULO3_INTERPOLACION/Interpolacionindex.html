Modulo 3: Interpolacion

Este modulo analiza la variacion de precios de productos basicos cuando solo existen datos dispersos durante el mes.

P(x) ≈ precio estimado en el dia x

La interpolacion permite reconstruir una curva aproximada de precios para estimar valores faltantes entre datos conocidos.

Contexto del ejercicio

¿De que estamos hablando?

En una situacion de crisis, los precios de alimentos basicos no siempre se registran todos los dias. A veces solo existen datos de algunos dias o semanas. Por eso se necesita interpolar para estimar precios faltantes.

En este modulo se trabaja con datos dispersos de precios durante un mes. El objetivo es reconstruir una curva continua que permita analizar como suben los precios en distintos escenarios.

Datos conocidos

Son pares ordenados de la forma (dia, precio), por ejemplo dia 10 y precio 13 Bs.

Dato faltante

Es el precio que se desea estimar en un dia donde no existe registro.

Curva interpolada

Es la curva que aproxima el comportamiento de los precios durante el mes.

Modelo general

Datos: (x₀,y₀), (x₁,y₁), ..., (xₙ,yₙ)

P(x) = precio interpolado

  • x: dia del mes que se desea analizar.
  • y: precio del producto en Bs.
  • P(x): precio estimado por interpolacion.
  • n: cantidad de datos conocidos.
  • Curva: comportamiento aproximado del precio.

Se analizaran tres escenarios de precios: estado ideal, crisis moderada y crisis critica. Cada escenario usara datos dispersos de precios durante el mes y se resolvera con los metodos de Lagrange, Newton y Splines cubicos.

Funciones usadas

Formulas de interpolacion aplicadas

Lagrange

P(x) = Σ yᵢ Lᵢ(x)

Usa todos los puntos conocidos para construir directamente el polinomio.

Newton Interpolacion

P(x) = a₀ + a₁(x-x₀) + a₂(x-x₀)(x-x₁) + ...

Usa diferencias divididas para calcular los coeficientes del polinomio.

Splines Cubicos

Sᵢ(x) = aᵢ + bᵢ(x-xᵢ) + cᵢ(x-xᵢ)² + dᵢ(x-xᵢ)³

Divide la curva en tramos para obtener una interpolacion mas suave.

Metodos obligatorios

1

Lagrange

Construye un polinomio usando todos los puntos conocidos. Es directo y facil de aplicar cuando hay pocos datos.

  • Usa todos los puntos.
  • No necesita tabla de diferencias.
  • Bueno para estimaciones puntuales.
Abrir Lagrange
2

Newton Interpolacion

Usa diferencias divididas para construir el polinomio interpolante. Es util cuando se agregan nuevos datos.

  • Usa diferencias divididas.
  • Permite organizar mejor los calculos.
  • Da el mismo polinomio que Lagrange.
Abrir Newton
3

Splines Cubicos

Une los puntos mediante polinomios cubicos por tramos. Produce una curva mas suave y estable visualmente.

  • Trabaja por intervalos.
  • Genera curvas suaves.
  • Reduce oscilaciones grandes.

Interpretacion de metodos

¿Que metodo puede ser mas estable o util?

Lagrange

Es util cuando se desea calcular directamente un precio faltante. Sin embargo, con muchos puntos puede presentar oscilaciones.

Newton

Es util para organizar mejor los datos mediante diferencias divididas. Es practico si se agregan nuevos registros.

Splines

Suele ser el mas estable visualmente, porque trabaja por tramos y genera una curva suave entre los puntos.

Conclusion

Para estimar un solo valor, Lagrange y Newton son utiles. Para mostrar una curva de precios mas suave y realista, Splines cubicos suele ser el metodo mas estable.
