Fundamento del Método SOR

El método SOR es una mejora del método de Gauss-Seidel.

Utiliza un parámetro de relajación ω que permite acelerar el proceso de convergencia.

Cuando se selecciona un valor adecuado de ω, el número de iteraciones puede reducirse significativamente.

Interpretación del Modelo Matemático

El sistema representa la distribución de combustibles desde las principales plantas de almacenamiento hacia los centros urbanos.

  • x₁: Gasolina distribuida (mil m³)
  • x₂: Diésel distribuido (mil m³)
  • x₃: GNV distribuido (mil m³)

Plantas consideradas

  • Planta Senkata (El Alto)
  • Planta Palmasola (Santa Cruz)
  • Planta Gualberto Villarroel (Cochabamba)

Escenarios del Proyecto

  • Estado Ideal: matriz diagonal dominante.
  • Crisis Moderada: bloqueo parcial de rutas.
  • Crisis Crítica: sistema mal condicionado y alta sensibilidad.
